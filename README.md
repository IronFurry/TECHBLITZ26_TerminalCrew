# ClinicOS — Doctor's Appointment & Scheduling System
## TechBlitz 2026 | Vibe Coding Domain

ClinicOS is a sleek, modern, and highly responsive web application designed for streamlined clinic management. Featuring dedicated portals for receptionists and doctors, it handles appointments, patient records, and integrated reminders seamlessly. 

### Features Include:
- Role-based Access Control (Receptionist & Doctor views)
- Custom Toast Notification System
- Fully Responsive Design (Mobile-first Bottom Tab Bar)
- Smooth page transitions and micro-interactions powered by Framer Motion
- Beautiful Light and Dark Mode theming
- Reusable UI Components based on a plain CSS design system

### Tech Stack
| Technology | Description |
|---|---|
| **React (Vite)** | Blazing fast frontend framework |
| **React Router** | Client-side routing with protected routes |
| **Framer Motion** | Declarative animations and page transitions |
| **Lucide React** | Consistent, crisp iconography |
| **Vanilla CSS** | Pure CSS theming via CSS variables |

### Setup Instructions

1. **Clone the repository:**
   ```bash
   git clone <repository_url>
   cd ClinicFlow
   ```
2. **Install dependencies:**
   ```bash
   npm install
   ```
3. **Environment Setup:**
   ```bash
   cp .env.example .env
   ```
   *(Fill in your Supabase keys based on the setup phase if moving beyond the mock data version)*
4. **Run the development server:**
   ```bash
   npm run dev
   ```
